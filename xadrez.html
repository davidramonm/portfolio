Projeto de Jogo de Xadrez em Java

Este projeto em Java consiste na implementação de um sistema para um jogo de xadrez, desenvolvido gradualmente com base em conceitos de Programação Orientada a Objetos (POO) e boas práticas de desenvolvimento, como encapsulamento, herança, polimorfismo e tratamento de exceções.

Estrutura do Projeto

Classes e Conceitos Iniciais

  • Position: Classe inicial que lida com a posição das peças no tabuleiro.
  • Board e Piece: Implementação da lógica básica do tabuleiro e peças de xadrez, utilizando matrizes como estrutura de dados.
  • ChessMatch e ChessPiece: Gerenciamento da partida de xadrez e das peças no contexto do jogo.

    • Lógica de Jogo

  • Movimentação das Peças: Métodos para movimentar as peças no tabuleiro, incluindo lógica de exceções e validações.
  • Captura e Turnos: Implementação de capturas de peças, alternância de turnos entre os jogadores e gerenciamento de peças capturadas.

    • Regras Específicas do Xadrez

  • Cheque e Cheque-Mate: Implementação da lógica para detectar e lidar com situações de cheque e cheque-mate.
  • Movimentos Especiais: Implementação dos movimentos especiais como roque, en passant, e promoção de peão.

    • Peças de Xadrez

  • Implementação das classes específicas para cada peça: (Torre, Rei, Rainha, Bispo, Cavalo, Peão), incluindo seus movimentos permitidos.

    • Interface de Usuário

  • Desenvolvimento de uma interface simples para exibir o tabuleiro e permitir a interação dos jogadores, incluindo a impressão de possíveis movimentos e peças capturadas.

    • O projeto finaliza com a implementação completa de todas as regras do xadrez, resultando em um jogo funcional que pode ser jogado através do terminal.
